Base URL

All the requests should start with the BASE_URL

          
    https://codebender-catalog.onrender.com

Available resources

  • Products
  • Details
  • Descriptions
  • Amount by category
  • Hot price and Newest products

Response format

Success response contains the requested data.

Error response contains an error property

              
    {
     
       "error": "Error message"
     
    }

Load products

GET /products returns an array of products

GET /products 

              
    [
     
      {
     
        "id": 1,
        "category": "phones",
        "itemId": "apple-iphone-7-32gb-black",
        "name": "Apple iPhone 7 32GB Black",
        "fullPrice": 400,
        "price": 375,
        "screen": "4.7 IPS",
        "capacity": "32GB",
        "color": "black",
        "ram": "2GB",
        "year": 2015,
        "image": "img/phones/apple-iphone-7/black/00.jpg"
      }, ...
     
    ]

GET /products?param1=value1&param2=value returns an array of items having: 

              
    product.param1 === value1 // for number or boolean
    &&
    product.param2 === 'value2' // for strings

GET params:
productType => phones, tablets, accessories
search => product.name.includes(search)
perPage => 4, 8, 16 products on page
page => page for pagination
sortBy => title, age, price

GET /products/productId returns an item having: 

            
    product.itemId === productId

Load product descriptions

GET /descriptions returns an array of product descriptions

GET /descriptions 

              
    [
     
      {
     
        "id": 1,
        "title": "Easy on the eyes. Easy to carry.",
        "text": ["some long text 1", "some long text 2"],
        "itemId": "apple-ipad-pro-256gb-silver",
     
      }, ...
     
    ]

GET /descriptions/itemId returns all descriptions for given item: 

            
    descriptions.itemId === itemId

Load amount of products by category

GET /stats returns an object of amount of products of each category

GET /stats 

              
     {
     
       "phones": 71,
       "tablets": 3,
       "accessories": 4,
     
     }

Load hot price and newest products

GET /homepage returns an object with 2 arrays hot price and newest products (10 items for each array)

GET /homepage 

              
    {
       "newest": [product1, product2, ...],
       "discount": [product1, product2, ...],
    }
